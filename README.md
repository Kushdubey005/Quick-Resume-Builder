# AI Resume Builder

A modern and responsive Resume Builder web application that allows users to create professional resumes quickly and efficiently.

## Features

- Create professional resumes in minutes
- User-friendly interface
- Real-time resume preview
- Responsive design
- Easy-to-edit resume sections
- Clean and modern UI
- Download-ready resume format
- Dopwnload Button Available
- Dark mode Option Available

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
- React Js

## Installation

1. Clone the repository

```bash
git clone https://github.com/Kushdubey005/Quick-Resume-Builder.git
```

2. Navigate to project folder

```bash
cd Quick-Resume-Builder
```

3. Open `index.html` in your browser
